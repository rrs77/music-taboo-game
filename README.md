## Quick Start

1. Open `Taboo.html` in a web browser
2. Enter a username (or school code for school accounts)
3. Select a game set
4. Start playing!

## Deployment

### Netlify
- Drag and drop the folder to [Netlify](https://app.netlify.com)
- Or use the Netlify CLI: `netlify deploy`

### GitHub Pages
- Push to GitHub
- Enable GitHub Pages in repository settings
- Set source to main branch

## File Structure

- `Taboo.html` - Main HTML file
- `app.js` - Application logic and UI rendering
- `game-logic.js` - Game state management and data persistence
- `game-data.js` - Default game sets and card data
- `styles.css` - Styling and animations
- `netlify.toml` - Netlify configuration
- `_redirects` - Netlify routing rules
